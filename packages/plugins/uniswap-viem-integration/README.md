# uniswap-viem-integration

EVM blockchain integration using viem for Uniswap V3 operations.

## Overview

This plugin provides reusable viem patterns for Uniswap V3 integrations including:

- Public and wallet client setup
- Chain-specific RPC configuration
- Contract read/write patterns
- Simulation-based quoting (QuoterV2)
- ERC20 approval flows
