# uniswap-risk-assessor

Risk analysis for Uniswap V3 including slippage, price impact, and impermanent loss.

## Overview

This plugin provides risk assessment tools for:

- Swap price impact analysis
- Slippage tolerance recommendations
- Impermanent loss calculations for LP positions
- MEV sandwich attack risk evaluation
- Fee APR vs IL break-even analysis

- `skills/risk` - Core risk assessment skill
