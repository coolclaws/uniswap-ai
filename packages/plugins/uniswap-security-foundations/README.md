# uniswap-security-foundations

Security guidance and audit checklists for Uniswap V3 integrations.

## Overview

This plugin provides security best practices for building safe Uniswap V3 integrations:

- Pre-execution security checks
- MEV protection strategies
- Token approval best practices
- Common vulnerability patterns and fixes
- Pre-deployment audit checklist
